About

This site is a concise reference for configuring Spring Boot microservices (Config Server, Eureka, Gateway, JWT, RabbitMQ, Databases, Resilience4j, etc.). The goal is practical, original documentation and example configs for developers.

What you'll find

  • Clear examples for application.properties / application.yml
  • Notes on common production settings and security tips
  • Links to related guides (Docker, monitoring, resilience)
